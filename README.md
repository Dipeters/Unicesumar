# Sistema de Agendamento de Consultas Médicas

Este projeto consiste em um sistema completo de agendamento de consultas médicas, desenvolvido como parte do Trabalho de Conclusão de Curso (TCC). 
A aplicação permite que pacientes realizem agendamentos, médicos gerenciem suas agendas e administradores controlem todas as operações do sistema.

## Tecnologias Utilizadas
Frontend: React  
Backend: Node.js + Express  
Banco de Dados: SQLite  
Segurança: Bcryptjs  

## Funcionalidades
Paciente: agendar consultas, editar perfil, histórico  
Médico: gerenciar consultas, agenda, bloquear horários  
Administrador: cadastrar médicos, gerenciar usuários, dashboard  

## Como Executar
1. Clonar repositório  
2. Instalar dependências do backend e frontend  
3. Executar `node server.js` no backend  
4. Executar `npm start` no frontend
